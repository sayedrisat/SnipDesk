# CodeVault

CodeVault is a dark, developer-focused snippet notebook built with Vite, React, Tailwind CSS, and Ace Editor.

The alpha release includes a compact sidebar, fast snippet search, reusable note cards, and a right-side editor drawer for creating or editing code notes.

## Tech Stack

- Vite
- React
- Tailwind CSS
- Ace Editor
- Lucide React icons

## Getting Started

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Alpha Scope

This release is intended as an early frontend alpha. Notes are currently stored in memory, so saved changes reset after a page refresh.

Planned improvements include localStorage persistence, note deletion, toast notifications, JSON import/export, and mobile sidebar refinements.
