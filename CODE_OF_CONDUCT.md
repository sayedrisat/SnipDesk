# Code of Conduct

## Our Pledge

We want CodeVault to be a welcoming project for contributors of all experience levels. Contributors are expected to communicate with patience, respect, and good faith.

## Expected Behavior

- Be respectful when reviewing code or discussing ideas.
- Give constructive feedback that focuses on the work.
- Keep issues and pull requests relevant to the project.
- Assume good intent when asking questions or offering suggestions.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discriminatory language or behavior
- Repeated off-topic comments
- Publishing private information without permission

## Reporting

If you notice behavior that does not match this code of conduct, please contact the repository maintainer or open a GitHub issue if appropriate.

Maintainers may remove comments, close issues, or block contributors who repeatedly violate these expectations.
